# UPGB Promotion Exam Prep 2025

A web-based study portal and mock test application for the UPGB Promotion Exam.

## Features
- **Study Mode**: Read all relevant circulars and policies directly in the browser (Text-based).
- **Mock Test**: A 20-question randomized mock test based on the 2025 syllabus.
- **Offline Ready**: All data is self-contained.

## Deployment (GitHub)
**IMPORTANT**: This folder `upgb-promotion-exam` is optimized for GitHub.

1.  **Do NOT upload the `unused_circulars_backup` folder** (it contains large files that will fail).
2.  Upload **ONLY** the contents of this `upgb-promotion-exam` folder.
3.  **Note**: The PDF backups are NOT included in this upload to keep it lightweight. The app relies entirely on the text-based study data.

## How to Run locally
Simply open `index.html` in your browser.
